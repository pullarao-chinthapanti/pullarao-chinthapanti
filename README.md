## Projects

**[workspace-admin-automator](https://github.com/pullarao-chinthapanti/workspace-admin-automator)**  
Automates Google Workspace onboarding, offboarding, licence audits, and account security using Google Admin SDK. Deployed on GCP Cloud Functions with Firestore audit logging.

**[activtrak-compliance-engine](https://github.com/pullarao-chinthapanti/activtrak-compliance-engine)**  
Pulls activity data via ActivTrak REST API, runs configurable compliance rules per user, and delivers automated weekly reports via AWS Lambda and SES.

Privacy-first UPI transaction tracker. Forward your bank SMS to a Telegram bot, get instant categorisation and monthly spend breakdown — fully serverless on AWS.
